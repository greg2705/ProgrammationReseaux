Bonjour, bienvenue dans notre projet de réseau.
Pour lancer le projet , veuillez executer les commandes suivantes sur le terminal :
   - make
   - ./Dazibao NomDuDomaine NuméroDePort

NomDuDomaine et NuméroDePort sont la pour les noeuds boostrap
Pour envoyer des messages , il suffit d'écrire sur l'entrée standard et d'appuyer sur entrée.
Pour quitter le programme , il suffit d'écrire 'quit' (sans les guillemets) sur l'entrée standard et d'appuyer sur entrée.
